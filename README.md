## Status Codes
The API here returns the following status codes:

| Status Code  | Description |
| ------------- | ------------- |
| 200  | Ok  |
| 201  | Created  |
| 404  | Bad Request  |
| 400  | Not Found  |
| 500  | Internal Server Error  |
